# Resumen Ejecutivo

El proyecto desarrolla un **Sistema Administrado de Contenidos para Pantallas SENA**, una plataforma web full stack que permite a cada centro de formación gestionar de forma autónoma los contenidos de sus pantallas de televisión.

### Características Principales:
- **Gestión Autónoma:** Roles diferenciados por sede (Administrador, Publicador, Auditor).
- **Asistencia de IA:** Integración de IA generativa para el apoyo en la creación de contenidos.
- **Hardware Dedicado:** Uso de MiniPC por pantalla configurados en modo kiosko.
- **Monitoreo IoT:** Sensores para el seguimiento de variables ambientales (temperatura, humedad, calidad del aire) en los ambientes de formación.

El objetivo principal es lograr la autonomía comunicacional, el uso eficiente de la infraestructura y la mejora constante de los ambientes de formación en cada sede del SENA.
