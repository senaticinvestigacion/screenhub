# Árbol de Objetivos

El árbol de objetivos representa la situación esperada al resolver las causas del problema central.

## Objetivo Central
**Desarrollar e implementar un Sistema Administrado de Contenidos para Pantallas SENA que permita a cada centro de formación gestionar de forma autónoma sus contenidos y monitorear sus ambientes de formación.**

## Medios (Cómo lo lograremos)
- **Software:** Módulos de usuarios con roles diferenciados, asistente de IA generativa y panel de programación por centro.
- **Hardware:** Selección, adquisición y configuración de MiniPC para cada pantalla.
- **IoT:** Instalación de sensores de temperatura, humedad y calidad del aire integrados a una API de monitoreo.

## Fines (Qué lograremos)
- **Directos:**
  - Sistemas de información dinámicos y actualizados por centro.
  - Monitoreo en tiempo real de condiciones ambientales.
  - Toma de decisiones basada en datos propios de cada sede.
- **Indirectos:**
  - Mejora de la comunicación institucional y la experiencia formativa.
  - Fortalecimiento de la autonomía tecnológica de cada sede.
  - Optimización del uso de la infraestructura existente.
