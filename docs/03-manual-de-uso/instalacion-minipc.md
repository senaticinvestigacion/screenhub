# Instalación de MiniPC

:::caution
**PENDIENTE POR DILIGENCIAR**

Pasos técnicos para configurar e instalar los MiniPCs en las pantallas de las sedes.
:::

## Requerimientos de Hardware
- Modelo de MiniPC recomendado.
- Conexiones necesarias (HDMI, Corriente, Red).

## Configuración de Software (Modo Kiosko)
1. Instalación del OS.
2. Configuración de arranque automático.
3. Enlace con la plataforma central.
