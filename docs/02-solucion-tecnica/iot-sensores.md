# Red de Sensores IoT

:::caution
**PENDIENTE POR DILIGENCIAR**

Detalle de la red de sensores para el monitoreo ambiental por centro.
:::

## Variables Monitoreadas
1. **Temperatura** (°C).
2. **Humedad Relativa** (%).
3. **Calidad del Aire** (CO2 / PM2.5).

## Hardware IoT
- **Microcontrolador**: [Ej. ESP32]
- **Protocolo de Comunicación**: [Ej. MQTT / HTTP]
- **Alimentación**: [Ej. USB 5V / PoE]
