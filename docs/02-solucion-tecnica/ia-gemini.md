# IA Generativa con Gemini

:::caution
**PENDIENTE POR DILIGENCIAR**

Explica cómo se integra la IA de Google Gemini para asistir a los usuarios en la creación de contenidos para las pantallas.
:::

## Casos de Uso
1. **Generación de Textos**: Creación automática de avisos institucionales.
2. **Optimización de Mensajes**: Ajuste de tono y longitud para pantallas.
3. **Asistencia Creativa**: Sugerencias de imágenes y layouts.

## Configuración de la API
- Modelo utilizado: [Ej. Gemini 1.5 Pro / Flash]
- Implementación del Prompt Engineering.
